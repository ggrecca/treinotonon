REFATORAÇÃO UI — ETAPA 1

Esta etapa separa do App.jsx três componentes visuais reutilizáveis:

- Card de indicador
- Seletor de treino no dashboard
- Seletor de treino na execução

Ela não altera regra de negócio, Supabase, histórico nem banco de dados.

COMO APLICAR

1. Descompacte o pacote na raiz do repositório. Isso criará os três arquivos
   dentro de src/components.
2. No terminal aberto na pasta do repositório, execute:

   git apply aplicar-refatoracao-ui-base.patch
   npm run test:run
   npm run typecheck
   npm run build

3. Se os três comandos terminarem sem erro:

   git add src/App.jsx src/components aplicar-refatoracao-ui-base.patch
   git commit -m "refactor: separar componentes visuais do app"
   git push origin main

O arquivo .patch pode ser apagado depois de o commit estar concluído.
